# expense-tracker
track income and expenses withcharts, categories, and balance calculations.
## Features

*   **Income & Expense Tracking:** Easily log your daily financial transactions.
*   **Dynamic Charts:** Visual breakdown of your spending habits and income categories.
*   **Category Management:** Organize transactions by custom or preset categories (e.g., Food, Rent, Salary).
*   **Live Balance Calculations:** Automatically calculates your current balance, total income, and total expenses.

## Tech Stack

*   **Frontend:** HTML5, CSS3, JavaScript (or React / Vue if applicable)
*   **Charts:** Chart.js (or D3.js / Recharts)
